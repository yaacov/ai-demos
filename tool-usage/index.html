Tool Use

A language model can only generate text — it lives inside a bubble with no access to the outside world. It can't check today's weather, look up a flight, or run a calculation on its own.

Tools break the bubble. Like a doctor writing a prescription instead of manufacturing the medicine, the model writes a structured request (a tool call) that an external system executes, then feeds the result back into the conversation.
