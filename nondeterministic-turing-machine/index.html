Nondeterministic Turing Machine

A standard Turing machine follows one path — each state and symbol leads to exactly one action. A nondeterministic Turing machine can branch: when multiple rules match, it explores all possibilities simultaneously.

If any branch reaches an accept state, the machine accepts. This model is central to complexity theory and the famous P vs NP problem — it defines the class NP.
