From Sentence Completion to Chat

A language model does one thing: predict the next word. Given "The sky is", it guesses "blue." It simply completes sentences — over and over.

So how does a simple text predictor become an assistant you can have a conversation with? The secret is the chat template — a structured format that teaches the model to take turns between user and assistant.
